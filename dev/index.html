Data Visualisations

This section contains a collection of data visualisations that I have created using d3.js.

Player comparison

This tool allows the user to compare the respective chance maps of two footballers from various European leagues. The size of each circle signifies the quality of the chance, while the colour shows the eventual outcome

Premier League week-by-week position line graph

This line graph shows the gameweek-by-gameweek league position of each team in the Premier League 2017/18 season. The user can select/unselect certain teams for comparison purposes as well as selecting/unselecting all.

Gambling in Great Britain

Interactive map showing the results of the Gambling Commission’s Health report on gambling. By clicking on a particular zone on the map, the user can see that region’s gambling statistics compared to those of the UK as a whole.

UK Parliament Constituencies

Map showing the current political landscape of the UK. The user can click on a particular constituency to discover the name of the standing MP, and the party they represent.
